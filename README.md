# Placement Prediction Dashboard

A machine learning dashboard that predicts whether a student is likely to be placed based on academic and skill-based indicators.

## Features

- Predicts placement status
- Uses Random Forest Classifier
- Interactive Streamlit dashboard
- Displays prediction probability
- Data preprocessing and feature engineering
- Feature importance analysis

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Dataset Features

- CGPA
- Coding skill score
- Communication skill score
- Logical reasoning score
- Internship count
- Projects count
- Certifications
- Mock interview score
- Attendance percentage
- Leadership score
- Extracurricular score

## Project Workflow

1. Data cleaning
2. Exploratory Data Analysis
3. Feature engineering
4. Model training
5. Accuracy evaluation
6. Streamlit deployment

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Business Value

Helps estimate student placement readiness using machine learning-based prediction.
